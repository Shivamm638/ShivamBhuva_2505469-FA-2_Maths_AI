# 🚀 Ultimate Crypto Volatility Dashboard  
### Mathematics for AI – Formative Assessment 2 (FA-2)

---

## 📌 Project Overview

The **Ultimate Crypto Volatility Dashboard** is a fully interactive web application built using Python and Streamlit.  

This project demonstrates how mathematical modelling concepts can be applied to simulate and analyse cryptocurrency price behaviour in real time.

The dashboard integrates:

- Data simulation
- Statistical analysis
- Mathematical functions
- Interactive visualization
- Real-time volatility modelling

The application connects Mathematics for AI theory with practical financial data visualization.

---

## 🎯 Project Objectives

This project was developed to:

- Apply mathematical concepts (sine waves, random noise, drift)
- Calculate volatility using standard deviation
- Implement moving averages
- Build interactive dashboards using Streamlit
- Visualize financial data professionally
- Deploy a working cloud-based web application

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- GitHub
- Streamlit Cloud

---

## 📊 Key Features

### 🔹 Multi-Cryptocurrency Support

The dashboard supports:

- Bitcoin (BTC)
- Ethereum (ETH)
- Binance Coin (BNB)
- Solana (SOL)

Users can dynamically switch between cryptocurrencies using a dropdown menu.

---

### 🔹 Mathematical Simulation Modes

Users can select:

1. **Real Market Data Mode**  
2. **Wave Simulation Mode** (Sine Function)  
3. **Random Volatility Mode** (Gaussian Noise)  
4. **Drift Adjustment** (Linear Trend)

This demonstrates real-world financial modelling using mathematics.

---

### 🔹 Advanced Visualizations

The application includes:

- 🕯️ Candlestick Chart (OHLC Analysis)
- 📈 Line Chart with Moving Averages (MA20 & MA50)
- 📦 Volume Bar Chart
- 🔥 Price Distribution Histogram
- 📊 Live Market Metrics Panel

All charts are interactive using Plotly.

---

## 📈 Mathematical Concepts Applied

| Concept | Application |
|----------|-------------|
| Standard Deviation | Volatility Index |
| Rolling Mean | Moving Averages |
| Sine Function | Wave Simulation |
| Random Noise | Market Fluctuations |
| Linear Function | Drift Modelling |

These mathematical techniques simulate real financial market behaviour.

---

## ⚙️ Interactive Controls

The sidebar allows users to:

- Select cryptocurrency
- Choose simulation mode
- Adjust amplitude (volatility strength)
- Adjust frequency (swing speed)
- Adjust drift (trend slope)

All charts update dynamically in real time.
